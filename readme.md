# 학교 시간표 알림이

###### 매 시간마다 알람 맞춰놓고 시간표확인하기 귀찮아서 만든 프로그램.
---
## 사용법
---
```
1. 'config' 폴더 안의 schedule.json 파일에서 시간표 정보를 입력하세요.(양식은 schedule_example.json 파일을 참고하세요.)
2. 실행한다.
3. 시간이 되면 자동으로 알림이 온다.
```
